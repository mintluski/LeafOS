📦 What is LeafOS?

LeafOS is a minimalist Linux distribution built from scratch for virtual machines, embedded systems, and old hardware (1 GB Celeron class). Uses musl libc + BusyBox with APK package manager (25,000+ Alpine packages).

⚡ Quick Start (QEMU)

# Download ISO from button above
qemu-system-x86_64 -m 512 -cdrom LeafOS-1-0-LTS.iso -nographic
# With networking
qemu-system-x86_64 -m 512 -cdrom LeafOS-1-0-LTS.iso -net nic,model=e1000 -net user

📥 Installation

Interactive (easy):

boot from ISO → type leafos-install → follow prompts → reboot

Automated: create /autoinstall.conf with:

AUTO_MODE=1
AUTO_DISK="/dev/sda"
AUTO_KEYBOARD="us"

📦 Package Management (APK)

apk update                     # Update index
apk add gcc musl-dev make      # C development
apk add python3 git nano       # Tools
apk search <name>              # Search packages

🔧 LeafOS Commands

leafos-help      # Show available commands
leafos-info      # System information
leafos-version   # Show version
leafos-install   # Run installer

🛠️ System Recovery

Select "LeafOS (Rescue Shell)" in GRUB, then:

mount /dev/sda1 /mnt
mount -t proc proc /mnt/proc
mount -t sysfs sysfs /mnt/sys
mount -o bind /dev /mnt/dev
chroot /mnt /bin/sh
